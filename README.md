# FlappyBird
A simple SDL-based flappy bird clone

To comile:
  Requirements:
    SDL 2.0.0 or newer
    SDL2 image v2.0.0 or newer
    
  Open parent folder in terminal and type make it will create a file called 'game' and you're done
